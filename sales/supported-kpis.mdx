## Special Behaviors

### Compounding KPIs: Churn and NDR

<Warning>These KPIs behave differently from all others.</Warning>

Unlike one-time rate improvements, reducing churn creates a **snowball effect**. Each year the retained base is larger, so the same churn rate saves more in absolute dollars.

The model runs a **two-futures simulation** comparing ARR trajectories with and without improvement:

- `ARR (no fix) = Base ARR x (1 - churn_rate)^year`
- `ARR (with fix) = Base ARR x (1 - improved_churn)^year`
- **COI = cumulative difference over the time horizon**

This makes churn and NDR particularly powerful KPIs for retention-focused products.

### Cost Reduction: Two Modes

Cost reduction KPIs can work in two modes:

<CardGroup cols={2}>
  <Card title="Reduce Cost" icon="arrow-down">
    Same output, lower budget. The impact shows up as **dollar savings** against the relevant budget line.
  </Card>
  <Card title="Increase Capacity" icon="arrow-up">
    Same budget, more output. The impact shows up as **additional units or throughput** without increasing spend.
  </Card>
</CardGroup>

The seller selects which mode fits the customer's situation. Both use the same underlying math but frame the output differently.
