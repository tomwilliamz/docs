---
title: "Glossary"
description: "Quick reference for terms used in Supercase business cases."
icon: "book"
---

| Term | Definition |
|------|-----------|
| **S0** | Current state - the metric as it stands today |
| **S1** | Achievable state - the realistic target for the metric |
| **a0** | Current adoption rate - % of the org following best practices today |
| **a1_diy** | DIY adoption ceiling - % that would adopt with a self-built solution |
| **a1_vendor** | Vendor adoption ceiling - % that would adopt with a vendor solution |
| **M0** | Financial pool - the dollar amount at stake for a given metric |
| **COI** | Cost of Inaction - the financial loss from not closing the S0-to-S1 gap |
| **P10 / P50 / P90** | Percentile bands from Monte Carlo simulation (conservative / expected / optimistic) |
| **Monte Carlo** | A simulation method that runs 1,000 scenarios with slight input variations to produce a range of outcomes |
| **Rollout profile** | The speed at which adoption ramps (fast / steady / slow) |
| **Proficiency curve** | Models the fact that even adopted users aren't fully productive immediately |
| **Time horizon (T)** | The number of quarters over which the model projects impact (typically 8-12) |
| **Net impact** | Total cumulative financial impact minus total costs |
| **Payback period** | The quarter where cumulative impact exceeds cumulative costs |
| **KPI** | Key Performance Indicator - the metric being modeled (win rate, churn, etc.) |
| **ARR** | Annual Recurring Revenue |
| **NDR** | Net Dollar Retention - measures expansion vs. contraction of existing accounts |
| **LTV:CAC** | Lifetime Value to Customer Acquisition Cost ratio |
| **Pipeline intensity** | M0 divided by the current rate (S0) - represents the total volume flowing through the metric |
